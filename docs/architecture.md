# 项目架构说明

## 1. 项目概述

本项目是一个基于 Electron + Vue 3 的 IELTS 智能练习系统，集成了阅读、听力、写作三大模块，并通过 LLM（DeepSeek API）提供智能评估和反馈。

## 3. 核心模块

### 3.1 Electron 主进程 (electron/)

| 模块 | 职责 |
|------|------|
| `main.js` | 应用入口、窗口管理、生命周期 |
| `preload.js` | IPC 桥接、安全上下文暴露 |
| `local-api-server.js` | 本地 HTTP API 服务 |

### 3.2 后端服务 (electron/services/)

| 服务 | 职责 |
|------|------|
| `llm-provider.js` | LLM 提供者管理（DeepSeek 等） |
| `evaluate.service.js` | 作文评估服务 |
| `essay.service.js` | 作文管理服务 |
| `topic.service.js` | 题目管理服务 |
| `settings.service.js` | 用户设置服务 |
| `config.service.js` | 应用配置服务 |
| `prompt.service.js` | Prompt 模板管理 |

### 3.3 数据访问层 (electron/db/)

| DAO | 职责 |
|-----|------|
| `topics.dao.js` | 题目 CRUD |
| `essays.dao.js` | 作文 CRUD |
| `settings.dao.js` | 设置 CRUD |
| `prompts.dao.js` | Prompt CRUD |

### 3.4 Vue 前端 (apps/writing-vue/)

| 模块 | 职责 |
|------|------|
| `views/ComposePage.vue` | 作文撰写页面 |
| `views/EvaluatingPage.vue` | 评估进度页面 |
| `views/ResultPage.vue` | 评估结果页面 |
| `views/HistoryPage.vue` | 历史记录页面 |
| `views/SettingsPage.vue` | 设置页面 |
| `components/NavBar.vue` | 导航栏组件 |
| `components/RadarChart.vue` | 雷达图组件 |

## 4. 数据流

### 4.1 写作评估流程

```
用户输入作文 → IPC 调用 → evaluate.service → 
LLM Provider → DeepSeek API → 返回评估结果 → 
存储到 SQLite → 更新前端显示
```

### 4.2 阅读练习流程

```
加载题库 → 用户答题 → 本地评分 → 
postMessage 通信 → 存储练习记录 → 
更新统计数据
```

## 5. 技术选型

| 层级 | 技术 | 原因 |
|------|------|------|
| 桌面框架 | Electron | 跨平台、原生能力 |
| 前端框架 | Vue 3 + Vite | 响应式、快速构建 |
| 数据库 | SQLite | 本地存储、轻量级 |
| LLM | DeepSeek API | 中文友好、性价比高 |
| IPC | preload + contextBridge | 安全隔离 |

## 6. 安全设计

- API Key 存储在用户设置中，不硬编码
- IPC 通过 preload.js 安全暴露
- 数据库文件位于用户数据目录
- 敏感操作需用户确认
